### Setup bot for deployment

1. Make sure you have node js and npm downloaded
2. Run Npm install to get dependencies 
3. Setup bot on Discord dev page (allow sending message and attach files)
4. Fill out information in .env.sample and rename the file to .env
5. NPM run register will add commands to the bot
6. Add bot to your server
7. NPM run start
8. Test the commands
